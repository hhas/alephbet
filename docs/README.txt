Aleph Bet is an open source continuation of the _Marathon 2_ game engine. It can play _Marathon_, _Marathon 2_, and _Marathon Infinity_, as well as third-party scenarios for _Marathon 2_, _Marathon Infinity_, and _Aleph One_.

For more information about Aleph Bet, see: <https://aleph-bet-marathon.github.io/>
